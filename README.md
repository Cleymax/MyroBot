## üí≠ How it works <a name = "working"></a>
The entire bot is written in java.

### Prerequisites
 - JDK >= 8
 - Gradle
 - Internet connection
 - Hypixel API key
 - Discord Application
 
### Installing

You can directly import the project with the Intellij Idea IDE.

+ Wrapper :
```bash
gradle wrapper --gradle-version 5.2.1 --distribution-type all
```

+ Run tests :

```bash
gradle test
```

+ Build the project :

```bash
gradle build
```

+ Run the Discord bot :

```bash
gradle run
```
